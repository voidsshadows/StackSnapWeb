# Getting Started

## Quick Installation

### Download StackSnap
1. Download `StackSnap.pkg` (for Mac).
2. Install it on your Mac.
3. Open `http://localhost:8080` to start.

## The Developer Way

If you prefer to build from source:

```bash
git clone https://github.com/Nesbesss/StackSnap.git
cd stacksnap
go build -o stacksnap ./cmd/stacksnap
./stacksnap
```

Visit `http://localhost:8080` to start the onboarding.
